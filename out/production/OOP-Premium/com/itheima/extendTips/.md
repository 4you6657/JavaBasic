权限修饰符
+ 用来限制类中成员（成员变量、成员方法、构造器、代码块...）能够被访问到的范围

权限修饰符种类以及各自作用

| 修饰符     |在本类中|同一个包下的其它类 |任意包下的子类|任意包下的任意类|\
| private   |✔      |               |           |               |\
| default   |✔      |✔              |           |               |\
| protected |✔      |✔              |✔          |               |\
| public    |✔      |✔              |✔          |✔              |
